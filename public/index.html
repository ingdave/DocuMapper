¿Qué hace esta herramienta?

DocuMapper permite tomar una lista de datos en Excel y "rellenar" automáticamente una plantilla de Word para generar decenas o cientos de documentos personalizados en segundos. Ideal para cuentas de cobro, contratos, informes o cartas masivas.

Preparación de Archivos
El Excel:
  • La primera fila debe tener los nombres de las columnas.
  • Evita celdas combinadas en los encabezados.
  • Cada fila debajo será un documento generado.
El Word (Plantilla):
  • Escribe etiquetas entre llaves: {Nombre}, {Identificacion}.
  • Pueden estar en tablas, encabezados o pies de página.
  • Usa el mismo nombre para que sea fácil identificarlo.
Proceso Paso a Paso

Arrastra el Excel primero y luego el Word, el sistema detectará automáticamente las columnas y las etiquetas de tu plantilla.

Aquí el programa relaciona los datos de Excel con las etiquetas de Word. Por ejemplo: "Donde dice, ej: {Nombre} en Word, pone lo que hay en la columna, ej: 'CLIENTE' de Excel". Debes mapear todas las etiquetas antes de continuar.

Elige cómo se llamarán los archivos. Puedes agregar un prefijo (ej: "Cuenta_") y seleccionar una columna de Excel para diferenciar cada archivo (ej: nombre de la persona).
Solución de Problemas Comunes

¿No aparecen mis etiquetas de Word?
Asegúrate de que no tengan espacios internos como { Nombre } (usa {Nombre}) y que el archivo sea formato .docx.

¿Los números salen raros?
En Excel, formatea las columnas de números o fechas como "Texto" si quieres que aparezcan exactamente como los ves.

¿Un campo sale vacío?
Si la celda en Excel está vacía, el programa dejará el espacio en blanco automáticamente.

Consejo: Usa siempre el botón "Vista Previa" antes de generar el lote completo para ahorrar tiempo y asegurarte de que todo encaje bien.
